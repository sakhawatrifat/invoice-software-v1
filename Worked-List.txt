Invoice Software New Features (January 2026 To March 2026):

======================

**Admin Panel Works**
01. CRM Module 
	-Manage Lead Source (Facebook, Google, Website...etc.)
	-Manage Lead (customer_full_name, email, phone, etc.)
	-Customer History Check by Name/Email/Phone, Also can check from Lead data list by selection specific lead.

02. HR Module
	-Manage Department
	-Manage Designation
	-User Details Page (Admin can check Specific user Attendance & Salary History From Details Page)
	-Staff Details Page (Admin can check Specific staff Attendance & Salary History From Details Page)
	-Attendance functionality. Example: Check In, Pause (Break), Check Out (Overtime hours Count, Forgot Option if not Checkout in time) including timezone based.
	-After check in, admin can track user location. (From where the staff signed in on duty)
	-Clock-In Popup: Auto show once daily between 8:00 AM – 5:00 PM.
	-Salary Generate (Also can add bonus, deduct salary as punishment)
	-HR Report -> Attendance Report (Everyting covered in one with details page)
	-Attendance Export PDF Option
	-HR Report -> Salary Report
	-Salary Export PDF Option
	-Individual Salary Pay Slip Export PDF Option

03. Expense Module
	-Manage Expense Category
	-Manage Expense with document record
	-Expense Report
	-Expense Export PDF Option

04. Manage Report
	-Net Profit Loss Report
	-Net Profit Loss Export PDF Option
	-Gross Profit Loss Export PDF Option

05. Modified Some Pages (Ticket & Payment Details pages updated for extra informations)

======================

**Staff Panel Works**
01. My Attendance
	-Attendance Report
	-Attendance Report Export PDF Option

02. My Salary
	-Salary Report
	-Salary Report Export PDF Option
	-Individual Salary Pay Slip Export PDF Option


======================

**Common Panel Works**
01. Sticky Note
	-User Can Assign Anyone single or multiple user
	-Reminder With Popup
	-Read/Unread Status for each user 
	-Reminder With Email Notification (Cron Job)
	-Activity log in note details page for each note (create,status change, update data)


02. Ticket Data Import
	-Ticket Live Data Fetch Via Flight API
	-Create Ticket Auto Form Fill-up by Selecting Flight Data Item  


03. Flight Status 
	-Flight Status Cross Check with System Data vs. Live API Data
	-Mail Sending Option to Client With Updated Flight Datetime and Updated Ticket PDF


04. Message (Internal Users Chat System)
	- Chat list with user photo, name, last message, time, and unread count
	- Online / last seen status with search by name
	- Send message with Enter + reply with quote preview
	- Attach files (size limit): image, video, audio, documents
	- Message status: Sent, Delivered, Read (double check)
	- Image popup viewer, video/audio player, other files as download link
	- 3-dot message menu: Reply, View time, Remove for me, Remove for everyone
	- Auto refresh for new messages
	- Load older messages on scroll up
	- Loading animation on send and delete actions
	- Fixed layout (only messages scroll), fully responsive
	- Floating chat widget with unread message badge
	- Widget includes same chat features (send, reply, attach, status, menu)
	- Widget controls: View All, Minimize, Back
	- Config settings: file size limit, refresh time, translations, local time display
	- Emoji picker in chatbox (similar to WhatsApp)
	- Message reaction system for each message
	- Message forwarding to other users or groups
	- Chat users nickname system


4.1 Group Chat System
	- Any user can create a group and add members
	- Supports all existing single chat features
	- Separate emoji reactions for each message by each user
	- Message forwarding supported in group chats
	- Group members nickname system
	- Group admin access/permissions to manage any member


05. Marketing
	-Email Marketing with attachment (image,pdf)
	-Sent Mail List Records With sent files
	-WhatsApp Marketing with attachment (image,pdf)
	-Sent WhatsApp Message Records With sent files

06. Missing Payment Data of Tickets: Show tickets whices are not stored in "Manage Payment"

07. Add Total Male, Female, Child, Infant With Icon & Number in all flight related pages and header upcomming flights drawer

======================

**Note: All new features included multi-language (English,Japanese) including PDF exports multilanguage supported.**
